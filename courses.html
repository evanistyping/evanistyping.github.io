Computer Mediated Communications

Introduction to the tools and technologies of modern responsive web design using HTML5 and CSS3; an introduction to JavaScript.

Computer programming I

Introduction to algorithms, elements of a programming language, including data types, packages, control structures, procedures and functions, basic input and output, arrays and records, text files, strings, variant records.

Computer Programming and Linear Data Structures

Abstract data types, generics, access or pointer types, dynamic memory allocation, recursion, concurrent programming and linear data structures, including stacks, lists and queues.

Data Structures

Design and implementation of hash tables, general trees, search trees, balanced trees and graphs. Comparison of sorting algorithms. Evaluation of the best data structure for a particular task.

Computer Systems I

Binary data representation, the assembly and machine language representations of executable code, memory hierarchies, process memory organization, and program linking.

Formal Lang & Functional Programming

Proof techniques. Introduction to the formal language classes and their machines, including regular languages and finite automata, context free languages and pushdown automata.
